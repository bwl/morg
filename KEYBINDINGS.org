| Windows        | =Cmd-w=     | Close window              |
| Editing        | =Cmd-z=     | Undo                      |

* Modifier Keys

| Name    | Mac Key     | Emacs | Example     |
|---------+-------------+-------+-------------|
| Super   | Command ⌘   | =s-=  | =s-s= save  |
| Meta    | Option ⌥    | =M-=  | =M-x= command |
| Control | Control ⌃   | =C-=  | =C-g= cancel |

Right Option (⌥) types special characters (€, —, etc.)
